Full description:
This application adds RTM Oka Express (MRC Deux-Montagnes) bus information to MonTransit.

RTM Oka Express (MRC Deux-Montagnes) serves Oka, Saint-Eustache, Saint-Joseph-du-Lac and Saint-Placide.

Once this application is installed, the MonTransit app will display bus information (schedule...).

This application only has a temporary icon and no screen: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by the Agence Métropolitaine de Transport.
https://www.amt.qc.ca/en/about/open-data

This application is free and open-source:
https://github.com/mtransitapps/ca-deux-montagnes-mrcdm-bus-android

This app is not related with the Agence Métropolitaine de Transport and RTM Oka Express (MRC Deux-Montagnes).
